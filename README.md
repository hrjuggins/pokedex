## Run locally

    ``` shell
    cd pokedex/
    npm run develop
    ```
## Build for development

```shell
cd pokedex/
npm run build
```

## To run tests

``` shell
cd pokedex/
<!-- to run end to end test -->
<!-- these tests will open in chrome -->
npm run test:e2e

<!-- to run unit tests -->
npm run test:unit
```

## Dependencies used

 - Jest and react testing library for unit testing
 - Cypress for end 2 end tests
 - Cypress axe for accessibility tests
