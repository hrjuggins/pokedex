## Run locally

```shell
npm run develop
```
## Build for development

```shell
npm run build
```

## To run tests

To run end to end tests (these tests will open in a chrome instance)

``` shell
npm run test:e2e
```

To run unit test

``` shell
npm run test:unit
```

## Dependencies used

 - Jest and react testing library for unit testing
 - Cypress for end 2 end tests
 - Cypress axe for accessibility tests
